Основные функциональные требования 
1)Cоздание и открытие файлов
 Возможность создания нового пустого документа.
 Возможность открытия существующих файлов (.txt, .rtf, .md).
 Поддержка функции "Открыть как" с выбором кодировки (UTF-8, ASCII и т.д.). 
Сохранение файлов 
2) Возможность сохранения документа в текущем формате.
 Поддержка функции "Сохранить как" с выбором кодировки.
 Функция автосохранения с настраиваемым интервалом.
 Редактирование текста
3) Возможность ввода, удаления и редактирования текста.
 Функции "Отмена" и "Повторить" для действий редактирования.
 Стандартные функции "Копировать", "Вырезать", "Вставить", "Выделить всё".
Форматирование текста
4) Настройка шрифта (тип, размер, стиль: жирный, курсив, подчёркнутый). 
Выравнивание текста (влево, по центру, вправо, по ширине).
Настройка цвета текста и фона.
 Настройка межстрочного интервала.
 Поиск и замена
5) Функция поиска текста с возможностью чувствительности к регистру.
Функция замены с возможностью "Заменить всё".
Дополнительные функции
Архитектура и дизайн
 Основные модули
 Главное окно: Контейнер для основного редактора, меню и панели инструментов. 
Редактор текста: Виджет, отвечающий за ввод и редактирование текста, реализованный на основе QPlainTextEdit или QTextEdit.
 Модуль форматирования: Управляет форматированием текста, включая шрифты, цвет и выравнивание. 
Модуль управления файлами: Реализует функции открытия, сохранения и экспорта документов. 
Модуль поиска и замены: Обеспечивает работу функций поиска и замены текста. 
Модуль подсветки синтаксиса: Обеспечивает подсветку для определённых типов файлов (например, Markdown, HTML). 
Структура меню и панели инструментов
 Файл: Новый, Открыть, Сохранить, Сохранить как, Экспорт в PDF, Печать, Выход. 
Правка: Отмена, Повторить, Копировать, Вырезать, Вставить, Выделить всё, Найти и заменить. 
Формат: Шрифт, Выравнивание, Цвет текста, Цвет фона. 
Вид: Вкладки, Показать счётчик слов, Режим чтения, Выбор темы. 
Горячие клавиши 
Ctrl + N: Новый документ
Ctrl + O: Открыть 
Ctrl + S: Сохранить 
Ctrl + P: Печать 
Ctrl + F: Поиск 
Ctrl + Z / Ctrl + Y: Отмена / Повторить 
Ctrl + A: Выделить всё
